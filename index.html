About this page

This is a reference aid, structured for easy reading on a phone. It is not a substitute for the doctor's signed prescription or the pharmacist's advice.

Every dose, timing, and medicine name should be verified against the original prescription before use. If anything looks wrong, defer to the doctor's written instruction.
